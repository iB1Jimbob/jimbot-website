About JimBOT

JimBOT is a multi purpose Discord bot built as a free time project. It features economy, fun, gambling, meme, moderation and other utility commands.
JimBOT is open source, so its source code is hosted publicly on GitHub for everyone to view.
JimBOT might not be in active development all the time, there are some updates though from time to time.
